## Транспортный справочник — строим JSON
В коде «Транспортного справочника» вы создавали JSON-объекты с ответами на запросы. Эти объекты хранятся в виде объектов стандартных типов C++: векторов, словарей или одиночных значений.

Поэтому для создания JSON-словаря вам нужно было использовать стандартный синтаксис инициализации контейнера `map` и не забывать про указание типа `json::Node`:

```cpp
json::Dict{
    {"stop_count"s, json::Node{total_stops}},
    {"unique_stop_count"s, json::Node{unique_stops}},
    {"route_length"s, json::Node{route_length}},
    {"curvature"s, json::Node{route_length / route_length_direct}},
} 
```

Кажется, достаточно просто и понятно. Но теперь представьте, как будет выглядеть создание сложного JSON с вложенными словарями и массивами. Вы быстро запутаетесь в разного рода скобочках, и читать эту конструкцию будет непросто.

В этом уроке вы напишете вспомогательную библиотеку для построения JSON с помощью цепочек вызовов методов. Похожим образом вы реализовывали построение SVG-объектов. Здесь вы используете специальные методы, чтобы поддержать иерархичность структуры, а также добавите проверки корректности: сначала на этапе выполнения программы, а затем, отчасти, и на этапе компиляции.
