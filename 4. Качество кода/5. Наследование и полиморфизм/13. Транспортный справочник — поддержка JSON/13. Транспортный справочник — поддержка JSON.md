### Транспортный справочник — поддержка JSON

Вернёмся к транспортному справочнику. Сейчас программа построчно считывает из stdin данные об автобусах и остановках, строит базу данных автобусных маршрутов, а затем выводит ответы на следующие запросы:

- названия остановок, через которые проходит заданный автобусный маршрут;

- названия автобусных маршрутов, которые проходят через заданную остановку.

Продолжите улучшать программу. Для этого используйте свои библиотеки JSON и SVG:

1. JSON-библиотеку, чтобы считывать структуру базы данных и запросы к справочнику в формате JSON. Выводить ответ тоже предстоит в формате JSON.

2. SVG-библиотеку для визуализации карты автобусных маршрутов.

Подробнее об этом в заданиях.
