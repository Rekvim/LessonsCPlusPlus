## Встроенные алгоритмы
Стандартный алгоритм `count` используется для подсчёта элементов. Он последовательно просматривает все элементы и подсчитывает равные искомому. В то же время контейнеры `set` и `map` имеют встроенный алгоритм подсчёта — метод `count`. Он учитывает внутреннее устройство этих контейнеров и выполняется значительно быстрее. Разница в скорости особенно заметна, когда количество элементов в контейнере велико.
```cpp
const set<string> menagerie = {"cat"s, "dog"s, "rat"s};
// Подсчитываем кошек стандартным алгоритмом
cout << count(menagerie.begin(), menagerie.end(), "cat"s) << endl;
// Используем встроенный алгоритм count.
cout << menagerie.count("cat"s) << endl;
// Вывод:
// 1
// 1 
```
Если есть возможность, выбирайте встроенный вариант алгоритма.
