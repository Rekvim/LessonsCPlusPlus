## Алгоритмы sort и reverse

В этом уроке вы познакомитесь со стандартными алгоритмами сортировки из библиотеки <algorithm>.

### Алгоритм sort
sort сортирует указанный контейнер или обычный массив по неубыванию:
```
sort(xs.begin(), xs.end()); // теперь элементы xs будут в возрастающем порядке 
```

Этот алгоритм умеет сортировать любые типы данных — строки, числа, пары строк и чисел:

```cpp
#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

using namespace std;

int main() {
    vector<int> numbers = {1, 5, 12, 3, 7};
    vector<string> continents = {"Eurasia"s, "Antarctica"s, "North America"s, "Australia"s};

    sort(numbers.begin(), numbers.end());
    sort(continents.begin(), continents.end());

    for (const auto& element: numbers) {
        cout << element << endl;
    }
    cout << endl;
    for (const auto& element: continents) {
        cout << element << endl;
    }
} 

```
Числа сортируются по возрастанию, строки — по алфавиту.
```
1
3
5
7
12

Antarctica
Australia
Eurasia
North America 
```
### Сортировка пар
Сортировать можно не только одиночные объекты, но и пары значений. Пары сортируются в лексикографическом порядке:

- Сначала пары упорядочиваются по первому элементу.
- Пары с одинаковым первым элементом упорядочиваются по её второму элементу.

В этой программе люди будут отсортированы по фамилии, а члены одной семьи — ещё и по имени:
```cpp
int main() {
    vector<pair<string, string>> people = {
        { "Potter"s, "Harry"s },
        { "Weasley"s, "Ron"s },
        { "Granger"s, "Hermione"s },
        { "Weasley"s, "Fred"s, },
        { "Hagrid"s, "Rubeus"s },
        { "Weasley"s, "George"s, },
        { "Malfoy"s, "Draco"s },
        { "Weasley"s, "Ginny"s },
    };

    sort(people.begin(), people.end());

    for (const auto& [surname, name] : people) {
        cout << surname << ' ' << name << endl;
    }
} 
```
Программа выведет:
```
Granger Hermione
Hagrid Rubeus
Malfoy Draco
Potter Harry
Weasley Fred
Weasley George
Weasley Ginny
Weasley Ron 
```
### Алгоритм reverse
```cpp
int main() {
    vector<int> numbers = {1, 5, 12, 3, 7};
    vector<string> continents = {"Eurasia"s, "Antarctica"s, "North America"s, "Australia"s};
    vector<pair<int, string>> numbers_with_names = {{2, "Two"s}, {1, "First"s}, {1, "One"s}, {3, "Three"s}};

    // sort заменили на reverse^
    reverse(numbers.begin(), numbers.end());
    reverse(continents.begin(), continents.end());
    reverse(numbers_with_names.begin(), numbers_with_names.end());

    for (const auto& element: numbers) {
        cout << element << endl;
    }
    cout << endl;
    for (const auto& element: continents) {
        cout << element << endl;
    }
    cout << endl;
    for (const auto& element: numbers_with_names) {
        cout << element.first << " " << element.second << endl;
    }
} 
```
Запустив программу, увидим, что все три вектора "развернулись":
```
7
3
12
5
1

Australia
North America
Antarctica
Eurasia

3 Three
1 One
1 First
2 Two 
```
Алгоритм `reverse` можно применять, чтобы отсортировать вектор в убывающем порядке. Для этого нужно вначале применить `sort`, а затем `reverse`:

- Сначала отсортировать вектор в прямом порядке, используя `sort`.
- Затем развернуть вектор алгоритмом `reverse`.

Эта программа сортирует названия животных в порядке убывания:
```cpp
int main() {
    vector<string> animals = {
        "penguin"s, "tiger"s, "leon"s, "zebra"s, "cat"
    };

    sort(animals.begin(), animals.end());
    reverse(animals.begin(), animals.end());

    for (const string& animal: animals) {
        cout << animal << " ";
    }

    // Выведет: zebra tiger penguin leon cat
} 
```
Такой способ сортировки по убыванию не самый эффективный — приходится переставлять элементы вектора дважды. Больше об алгоритме `sort` будет далее в курсе. Вы увидите, как получить любой желаемый порядок сортировки, используя только этот алгоритм.
