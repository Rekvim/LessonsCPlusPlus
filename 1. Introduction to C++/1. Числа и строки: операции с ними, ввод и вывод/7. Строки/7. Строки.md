## Строки

Редко встречаются программы, в которых нужно обрабатывать только целые числа. Строки — ещё один часто используемый тип данных. В С++ ему соответствует тип `string`. 

Чтобы создать строковую переменную, объявите переменную `news` типа `string` и положите в неё значение `"Great news!"`:
```cpp
string news = "Great news!"s;
// Переменная news содержит текст Great news!
```
Текст внутри двойных кавычек, за которыми следует символ `s`, задаёт строковый литерал типа `string`. Если пропустить букву `s` после кавычек, получится строковый литерал другого типа — его C++ унаследовал от своего предшественника, языка C.

Переменную типа `string` можно инициализировать с использованием обоих видов строковых литералов. Но инициализация на основе `""s`-литералов предпочтительнее, так как в них могут быть любые символы. Ещё такая инициализация помогает компилятору сгенерировать более быстрый код. 

В строковых литералах языка C символ с кодом 0 обозначает конец строки, поэтому строку, содержащую такие символы, объявить будет сложнее. В уроках и заданиях Практикума будут использоваться в основном `""s`-литералы.

Подключите библиотеку, где определён тип `string`:
```cpp
#include <string> 
Код будет выглядеть так:

#include <string>

using namespace std;

int main() {
        string news = "Great news!"s;
        return 0;
}
```
Внутри функции `main` инструкция `return 0` завершает работу программы с кодом ошибки, равным нулю. Это говорит об успешном завершении программы. Ненулевой код свидетельствует об ошибке. В простых программах вроде этой и “Hello, friend!” команда `return` в функции `main` необязательна. Однако в некоторых командах хорошим тоном считается явное использовании `return 0` внутри `main`. В примерах и задачах Практикума мы будем опускать `return` в функции `main` для краткости.

Если не подключить библиотеку `<string>`, компилятор выдаст ошибку:
```
string.cpp:4:5: error: ‘string’ was not declared in this scope
     string hello = "Great news!"s;
     ^~~~~~
```
Некоторые библиотеки, такие как `<iostream>`, автоматически подключают `<string>`, но лучше делать это самостоятельно и явно.

Если просто объявить строковую переменную, не указав начальное значение, в ней будет содержаться пустая строка. Этим тип `string` отличается от `int`, у которого значение неинициализированной переменной будет неопределённым:

```cpp
int age;      // В переменной age будет содержаться мусор (неопределённое значение)
string name;  // В переменной name будет пустая строка 
```

Для строк определена операция конкатенации, то есть склеивания содержимого двух строк в одну. Для этого используется операция `+`:

```cpp
string great = "Great "s;
string news = "news!"s;
string design = great + news; // Получается "Great news!" 
```

Чтобы просто последовательно вывести две строки, склеивать их не нужно. Просто выведите строки одну за другой.

```
cout << great << news; 
```

Как и у чисел, у строк есть операция `+=`. Она дописывает содержимое второй строки в конец первой:

```
string great = "Great "s;
string news = "news!"s;
string design;
design += great;
design += news;
// Теперь в переменной design содержится строка "Great news!"
```

При считывании строки из `cin` она считывается до первого символа-разделителя: пробела, табуляции или перевода строки. При этом пробелы между словами не считываются:

```cpp
string name;
string surname;
cout << "Enter your name:"s << endl;
cin >> name >> surname;
cout << "Hello, "s << name << " "s << surname << endl; 
```

Если пользователь введёт строку `John Doe`, то слово `John` будет прочитано в переменную `name`, а слово `Doe` — в переменную `surname`.

Чтобы считать всю строку, а не только текущее слово, используют функцию `getline`. Она считывает символы из потока ввода в строку, пока не встретится символ конца строки либо не произойдёт ошибка. 

Эта программа считывает две строки: имя и адрес. Вводимые строки могут состоять из нескольких слов:
```
string full_name, address;
getline(cin, full_name);
getline(cin, address);
cout << "Hello, "s << full_name << " from "s << address << endl; 
```
Введём следующие данные:
```
Harry Potter
4 Privet Drive, Little Whinging, Great Britain 
```
Программа выведет:
```
Hello, Harry Potter from 4 Privet Drive, Little Whinging, Great Britain 
```
В этой программе `cin` выступает в роли параметра функции `getline`. В C++ кроме `cin` есть и другие потоки ввода, с которыми может работать `getline`. О них вы узнаете в будущих уроках.
