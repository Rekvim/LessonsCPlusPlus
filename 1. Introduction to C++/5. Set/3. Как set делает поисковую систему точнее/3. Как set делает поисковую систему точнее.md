## Как set делает поисковую систему точнее
В прошлом уроке вы превратили вектор во множество, используя вставку внутри цикла `for`. Другой способ это сделать — использовать методы `begin()` и `end()`:
```cpp
set<string> MakeSet(vector<string> query_words) {
    set<string> s(query_words.begin(), query_words.end());
    return s;
} 
```

Код стал гораздо компактнее.

Создать вектор из множества тоже легко: `vector<string> v(s.begin(), s.end())`. Методы `begin()` и `end()` будут детально разобраны далее в курсе.

Умение превращать другие типы данных во множество поможет сделать вашу поисковую систему лучше.
