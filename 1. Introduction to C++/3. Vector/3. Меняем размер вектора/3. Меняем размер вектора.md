## Меняем размер вектора

Вставка элементов в вектор методом `push_back` позволяет увеличить его размер. Но если добавлены лишние элементы, то для их удаления нужен специальный метод. Ещё порой  хочется добавить много элементов за раз, а использовать для такого `push_back` не только неудобно, но и неэффективно.

### Меняем размер

Чтобы изменить размер вектора, используют метод resize. В самом простом варианте resize принимает один аргумент — новый размер вектора. Если значение этого аргумента меньше текущего, лишние элементы отбрасываются:
```cpp
vector<int> car_velocities = {60, 53, 67, 19, 77, 59};
car_velocities.resize(4);
// Теперь размер вектора — 4, содержимое: 60, 53, 67, 19 
```
Если новый размер больше текущего, в конец вектора добавятся недостающие элементы. Это будут стандартные «пустые» объекты соответствующего типа. Когда вектор состоит из чисел, добавляются нули, когда из строк — пустые строки, а когда из векторов — пустые векторы.

Так можно изменить размер вектора, который хранит возраст пропавших животных:

```cpp
vector<int> lost_pet_ages = {1, 8, 2, 1, 3, 10};
lost_pet_ages.resize(10);
// Теперь размер вектора — 10, содержимое: 1, 8, 2, 1, 3, 10, 0, 0, 0, 0
```
Когда важно, чтобы новые элементы имели конкретное значение, его передают в качестве второго аргумента. Например, в случае с возрастами животных 0 — вполне возможное значение, поэтому используем -1, чтобы отметить элементы, которые ещё не заполнены:
```cpp
vector<int> lost_pet_ages = {1, 8, 2, 1, 3, 10};
lost_pet_ages.resize(10, -1);  // Читается так: вектор должен иметь размер 10.
                               // Если нужно добавить новые элементы,
                               // это должны быть числа -1.
// Теперь размер вектора — 10, содержимое: 1, 8, 2, 1, 3, 10, -1, -1, -1, -1. 
```
Если требуется создать вектор, состоящий из некоторого количества одинаковых элементов, нужный размер можно указать при создании:

```cpp
size_t size;
cin >> size;
vector<string> lost_pet_names(size, "N/A"s); 
// Вектор из введённого количества строк N/A.
// Эта аббревиатура используется для замещения отсутствующих данных.
// Задать их можно позже. 
```

<div class="callout">
Значения `size` и `"N/A"s`, указанные в скобках после имени переменной, передаются в конструктор — специальный метод вектора, который инициализирует объект при объявлении переменной. В данном случае параметры задают начальный размер вектора и значение, которое должны иметь все его элементы. Подобный конструктор есть не только у `vector`, но и у типа `string`. С этим понятием вы ближе познакомитесь в следующих уроках.
</div>

Можно создавать вектор, не указывая никаких дополнительных параметров:
```cpp
vector<string> elephant_wings;

cout << "Elephant has "s << elephant_wings.size() << " wings."s << endl;
// Вывод: Elephant has 0 wings. 
```
В этом случае создаётся пустой вектор. Может показаться, что он не нужен. Но это не так: в пустой вектор можно добавить значений, и он станет непустым:
```cpp
vector<string> elephant_friends;

string friend_name;
cout << "Enter friend names or \"end\" to exit:"s << endl;

// Используем специальный приём: чтение из cin как условие цикла.
// При этом цикл будет работать, пока чтения проходят успешно.
// Если данные закончились или не могут быть прочитаны по другой
// причине, цикл остановится.
// Подобное выражение с cin можно использовать в условии if. Значение
// условия будет истинным, если чтение было успешным. А если нет — ложным.
while (cin >> friend_name) {
    // Завершаем цикл, если прочитано end.
    if (friend_name == "end"s) {
        break;
    }

    // Добавляем прочитанное имя в вектор.
    elephant_friends.push_back(friend_name);
}

// Выводим информацию о том, сколько элементов добавили в вектор
cout << "Elephant has "s << elephant_friends.size() << " friends."s << endl; 
```
