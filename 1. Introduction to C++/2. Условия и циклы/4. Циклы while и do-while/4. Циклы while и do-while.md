## Циклы while и do-while
Чтобы программа выполняла действие, пока есть некоторое условие, используют цикл `while`.

```cpp
while (в холодильнике торт) {  
    ходить к холодильнику;
}
```

Допустим, пользователь вводит число n. Так с циклом `while` можно посчитать сумму чисел от 1 до n.
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;
    int sum = 0;
    int i = 1;
    while (i <= n) {
        sum += i;
        ++i;
    }
    cout << sum << endl;
}
```
В следующей программе цикл `while` используется, чтобы найти НОД — наибольший общий делитель двух натуральных чисел `a` и `b`. Тело цикла повторяется, пока одна из переменных `a` и `b` не станет равной нулю. Вторая переменная будет содержать значение НОД:
```cpp
#include <iostream>

using namespace std;

int main() {
    int a, b;
    cin >> a >> b;
    // Числа a и b должны быть натуральными
    while (b != 0) {
        int c = b;
        b = a % b;
        a = c;
    }
    cout << a << endl;
} 
```

У `while` есть аналог — цикл `do-while`:

```cpp
do {  
    ходить к холодильнику;
} while (в холодильнике торт); 
```

В цикле `do-while` выполняет тело цикла хотя бы один раз, так как условие продолжения цикла выполняется в конце. В цикле `while` условие продолжения цикла проверяется в самом начале, поэтому тело цикла может не выполниться ни разу, если условие будет изначально ложным.

Посмотрите на фрагмент кода интерактивной игры, в которой пользователь угадывает число:
```cpp
int secret = 5; // Угадываемое число
int answer;     // Ответ пользователя
do {  
    cout << "Guess the number: "s << endl;  
    cin >> answer;
} while (answer != secret);
cout << "You are right!"s << endl; 
```
Пользователь вводит число, а код повторяет операцию, пока число не будет угадано. Если пользователь угадает число, программа сообщит ему: “You are right!”.

Разобравшись с циклом `while`, можно переходить к циклу `for`.
