## Зачем нужны функции
Программа на C++ состоит как минимум из одной функции — main(). С неё начинается работа программы. Любая нетривиальная программа, как правило, помимо main содержит и другие функции. Для этого есть ряд причин.

- Функции облегчают чтение и понимание кода. Встретив в коде функцию, достаточно прочитать её имя, чтобы понять, что она сделает;

- Написанную однажды функцию можно вызвать многократно. Функции позволяют переиспользовать код и избежать его дублирования. А значит, сокращают объём кода и количество возможных ошибок;

- Функции делят код на логические фрагменты и облегчают поиск проблем. Нормальное состояние разработчика — искать ошибки в исходном коде программы. Когда программа разбита на «строительные блоки» и каждый из них можно протестировать отдельно, отладить код проще;

- Любая функция умеет вызывать другие функции. Это позволяет разбить большую задачу на несколько более простых подзадач;

- Функции — это просто красиво! Потому что коротко и ясно.

Функция должна решать одну небольшую задачу, иметь понятное имя и имена входных параметров. Потренируйтесь в написании функций.
